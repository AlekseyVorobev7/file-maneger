# File Manager

Консольный файловый менеджер на Python для управления файлами и директориями в изолированной рабочей среде.

## Как это работает

Приложение предоставляет интерфейс командной строки для выполнения основных операций с файловой системой. Все операции ограничены одной рабочей директорией (по умолчанию `./workspace`), что обеспечивает безопасность и предотвращает изменение системных файлов.

## Архитектура

- **Core** — логика навигации (`navigator.py`), файловых операций (`file_ops.py`) и проверки безопасности (`security.py`)
- **UI** — консольный интерфейс (`cli.py`) для обработки команд пользователя
- **Workspace** — изолированная директория для всех файловых операций

## Доступные команды

| Команда | Описание |
|---------|----------|
| `ls` | Список файлов и директорий |
| `cd <path>` | Переход в директорию |
| `mkdir <name>` | Создать директорию |
| `touch <name>` | Создать файл |
| `cat <file>` | Показать содержимое файла |
| `write <file> <text>` | Записать текст в файл |
| `rm <name>` | Удалить файл или директорию |
| `cp <src> <dst>` | Копировать файл |
| `mv <src> <dst>` | Переместить файл |
| `pwd` | Показать текущий путь |
| `exit` / `quit` | Выход из программы |

Конфигурация рабочей директории задаётся в файле `config.json`.
